# Siglent SSG3000X SCPI Socket Test

Ein einfacher Python-Test, um die SCPI-over-Socket-Verbindung zu einem Siglent SSG3000X Signalgenerator zu prüfen.

## Voraussetzungen

- Ubuntu/Linux
- Python 3
- LAN-Verbindung zum SSG3000X
- gültige IP-Adresse des Signalgenerators

## Eigene IP-Adresse anzeigen
ip a

ping IP
nc -vz IP PORT
